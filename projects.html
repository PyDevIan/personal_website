Fridge Alert Notification System
Problem:

Industrial fridges must maintain strict temperature control for food safety. When a temperature breach is recorded by a fridge monitoring device, delays in response can cause product loss, compliance issues, or operational downtime. There was no automated alert system in place to track, assign, and escalate these events in real time.

Solution:

I developed an automated notification agent that monitors a dedicated inbox for alert emails sent by the fridge's industrial recorder. Upon detecting a new alert, it initiates a message loop in Microsoft Teams, notifying key roles (Factory Director, Head Mechanic, Automation Expert).

A FastAPI-powered web interface allows technicians to claim or acknowledge the alert, recording their action. If no one responds within a time window, the agent escalates the alert by re-notifying the team. The loop continues until a resolution email is detected (same device ID, alert marked inactive).

All alerts, acknowledgments, and responsible personnel are logged in a structured CSV file for auditability and trend analysis.
